# 📊 Sales Data Analysis using Python & SQL

## 🔹 Overview
This project demonstrates how Python integrates with SQL to perform data analysis and generate insights.

## 🔹 Features
- Created SQL database using SQLite
- Connected Python with database using sqlite3
- Executed SQL queries for filtering and aggregation
- Performed data analysis using Pandas
- Exported results to CSV and Excel files

## 🔹 Tech Stack
- Python
- SQLite
- Pandas
- openpyxl

## 🔹 Output
- Total Revenue Calculation
- Category-wise Sales Analysis
- Top Selling Products

python main.py
